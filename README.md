# 大数据重点技术栈

<p>
<a href=""><img src="picture/bigdata_architecture.png" align="center"></a>
<p/>

- 采集层：提供实时和离线数据采集功能

- 传输层：将采集层数据传入到统一存储层

- 存储层：提供统一数据资源存储资源池，同时拥有OLAP和 OLTP功能

- 计算层：提供离线和实时计算功能

- 工具层：直接和计算层、存储层进行交互，实现业务能力输出

- 业务层：大数据能力输出，实现业务价值
